# <h1 id="2">基本定義：</h1>

會計：經濟事項交易加以認定，**衡量**，**紀錄**，**分類**，**彙總**成有意義財務資訊，並**傳遞**給相關人士的過程

報表VS報導： 

主要報表 = 財務狀況表 + 綜合損益表 + 現金流量表 + 股東權益變動表

報表 = 主要報表 + 附註

報導 = 財務報表 + 營運計畫及公開說明等資訊

##### 衡量財務報表之要素

-- 衡量財務狀況之要素(財務狀況表)

-- 衡量經營績效之要素(綜合損益表)

##### 財務會計的五大要素

- 資產：**未來**具備**經濟效益**的經濟交易事項。

- 費用： 已經**失去**任何**經濟效益**的事項。

- 負債： 是指因為過去的交易或事件，使得**未來必須要犠牲**的經濟利益。這些現在獲得的資產，有在未來償還的**義務**。

- 收入： 收入，是指企業在**一定的期間**內，由**正常經營活動**所產生的**經濟利益**的**流入**的總額。該流人僅指引起權益增加的部分，而**不包括企業投資者出資引起的部分**。 

- 業主權益：是指企業所有權人對企業**剩餘資產**的**請求權**；如果用數字顯示，則是企業**資產**數額扣除所有**負債**後的**剩餘**部分。

#### (可參考的題型：P45, 6)


### 財務報表認列之條件

-- 該項目成本具備衡量性

-- 該項目未來經濟效益可能流入或流出企業

### 會計最低基本假設與限制

#### 設準：

1.)經濟個體假設：公司是與業主獨立分開的法人

2.)繼續經營假設：假設公司不會倒永續經營的情況

3.)貨幣單位假設：所衡量之貨幣單位為穩定可衡量且波動不大

4.)會計期間假設：設準 (2) 為了在永續經營情況下便於表達即時資訊，劃分報導期間

(註：期間可能其間可能是是年、月、日，但須注意成本效益原則)

5.)收費損益配合原則：

#### 驗證基準：

1.)成本效益之均衡：做這樣的帳，是否符合成本效益？   (安捏剛ㄟ合?

2.)攸關性：這個帳是否讓你好預測，或確認他的價值？

3.)忠實表述： 這個帳是否完整誠實的表達？ (完整性、中立性、沒錯誤


##### 品質再加強：(超過100分的好帳帳)

1.)可比較性：能夠辨識具備相同產業的相似處差異處

2.)時效性： 這個資訊具備快速 影響決策  (成為一個好棒棒的資訊內線交易者)

3.)可了解性： 這個帳是否連一般股東都看得懂？或者只是自HIGH

4.)可驗證性： 這個帳是否可以用有系統的方式去衡量，而確定無錯誤或偏差值。

# <h1 id="5">損益權益變動的部分：</h1>

## 股東權益定義：

### 為資產減去負債後所剩餘之價值：簡單分成三類：

- 投入資本： 股本 - 普通股、特別股 ， 資本公積 發行股票之溢折價 (請參閱老師line題目 https://goo.gl/LHptqN)

- 保留盈餘： 已指撥 - 法定盈餘公積 ， 未指撥 - 可自由分配之盈餘

- 其他權益項目： 未實現重估價值， 金融資產未實現損益


## 何謂保留盈餘：

是指公司歷年累積之純益，**未**以現金或其它資產方式分配給股東、轉為資本或資本公積者。

### 保留盈餘的統計方法為：
  保留盈餘 = 期初保留盈餘 + 凈利 - 派發之股息

### 用途通常為：

  - 將資金給公司本身再投資

  - 回購庫藏股

  - 投資其他新企業

### 保留盈餘又分成兩類：未指撥、已指撥

#### 已指撥的有：
 1.) 法定盈餘公積： 法律規定將盈餘之10%轉入公司之公積金，超過總資本50%可以不用再注入資金。

 2.) 自願提撥：意外損失準備、擴充廠房準備、平均股利準備

 3.) 契約規定：償債(基金)準備、特別股贖回準備

#### 未指撥

可自由分配之資金，可選擇發放股利或者保留再投資
  
### 練習題：

嘉瑋公司以每股16元發行普通股股票 10,000股，普通股面額為10元，於 12/1 收到全數現金

12 / 1

>　　普通股股本　　          100,000

>前期之損益調整　　10,000

>　　保留盈餘　　　10,000


# <h1 id="3" >現值與終值：</h1>

### 何謂"現值"：

現值定義為： 在給定的時刻，一次（或多次）該時刻之後發生的現金（現金流）在該時刻的價值（總和）稱為現值。

說白了就是把所有未來的價值轉換為現在的價值，比如說你看到了未來10年後的自己是一位籃球員，那麼就要推導這段時間

到底發生了什麼是那你成為那樣的人？最重要的關鍵因素是：發生變化的期間(期間)以及變化率(利率)。

### 何謂"終值"：

終值又稱將來值或本利和，是指現在一定量的資金在將來某個時點上的價值。

說白了就是如果依照你現行的規劃，去估計到未來時會變成什麼樣的人？最重要的關鍵因素也是一樣：發生變化的期間(期間)以及預期變化率(利率)。

### 何謂複利：

定義為：每次計算利息時，如有前期未付之利息，則應加入本金一併計息，及所謂"利上生利"。

公式定義為： a = p(1 + i) ** n  (**等於n次方)

### 何謂"年金"：

定義為：連續數期，每期支付或收取一定之金額，又分成兩種：

- 普通年金：期末支付。

- 到期年金：期初支付。(通常會多普通年金一期)

### 特徵：

依照課本模型，有分成三種特徵：

- 每期相隔時間相同

- 每期支付金額相同

- 每期計息之利率相同

(註：課本之定義但現實絕對不一定是如此。)

### 練習題(歡迎大家贊助提供)：

### 現在存款$85,000 年利率12% 每半年複利一次，求5年期滿本利和？

講解：此題為複利終值 用現有的價值去估計未來的價值，並帶有複利。

1.每期利率 i = 12% / 2 = 6% 計息期數 n = 2 * 5 = 10

2.a = p * a(n,i) = 272,606

### 我家希望能在六年後擁有一台PS5電動遊樂器，價值35,000，市場利率為8%，每年複利一次，請問我現在該存多少錢？

講解：此題為複利現值，利用未來已經發生的價值，去倒推回來現在的價值。

i = 8%, n = 6, a = $35,000

p = a * p(6,8%) = $35,000 * 0.630170 = 22,056

### 嘉瑋公司租用機器一台，租期六年，每年租金為35,000於年初支付，設利率為9%並每年複利一次，求設備租金的年金現值。

講解：此題目為計算年金現值，因為是在年初支付(到期年金)，所以我們先加上160,000的本金就變成年底的普通年金囉。

(注意：到期年金不難，只要加上最初的本金就變成了年底支付的普通年金。當然，因為是年底，所以總年數會少一年。)

i = 9%, n = 6 - 1

P = 35,000 * P(5,9%) + 35,000 = 35,000 + 35,000 * 3.889651 = 171,137.78

### 嘉瑋公司購買機器一台，以面額120,000 之兩年期不付息票據抵付，若市場利率10%，則該票據現值為何？

講解：此題目之票據雖不付息，卻包含了兩年的時間價值，必須要把那些消失的兩年找回來。

120,000 * p(2, 10%) = 120,000 * 0.826446 = 99,174

(做分錄需注意票據的折溢價唷，***應付票據折價***為應付票據面額的減項。)

### 張君雅計畫於X4年存入存款供大學之用，預定X8年後入學，自X8年8月1日起每年預計提款50,000共四年，若銀行利率為8%，每年複利一次，則張君雅於X4年需存多少錢？

講解：X8年之後都是提款50,000元，代表是計算年金的現值，X4年沒有提領，所以會繼續利滾利。

P = 50,000 * P(4,8%) = 50,000 * 3.312127 = 165,606.35

算出的答案為X8年的價值，現在要回推到X4年的價值，期中沒有提領，沒有存入，只是單筆價值的複利。

現在計算X4年的現值：

165,606.35 * p(3,8%) = 165,606.35 * 0.793832 = 131,464


<h1 id="4">現金與應收款項：</h1>

### 應收帳款的定義為：

應收帳款乃企業對顧客或他人之貨幣，商品或勞務的 **請求權** 。大致分為兩種：

#### 因主要營業活動產生者：

- 無債權憑證 => 應收帳款

- 有債權憑證 => 應收票據

#### 與非主要營業活動產生者：

- 金額小列為其他應收款

- 依性質劃分可列為應收租金，應收利息，應收退稅款......等等

### 應收帳款認列時機：

- 時機： 銷貨交易完成，商品所有權轉移買方或提供勞務後認列。

- 入帳金額： **商業折扣**直接認列折扣後的數字，**現金折扣**則依照**總額法**、**淨額法** 做認列，顧客若未取得折扣則以[**顧客位取得折扣**]分錄來認列其他收入。

### 總額法(假設顧客不會先付，如果先付是獎勵值，可以獲得折扣)：

賒銷：

>應收帳款

>　　銷貨收入

收款：

>現金

>　　銷貨折扣

>　　應收帳款

超過時間才收到款：

>現金

>　　應收帳款


### 淨額法(假設顧客是最棒的，會準時先拿折扣，如果沒拿就算我賺到的)：

賒銷：

>應收帳款

>　　銷貨收入

收款：

>現金

>　　銷貨折扣

>　　應收帳款



### 呆帳的定義為：

應收帳款中無法收回的部分，歸類為費用。

### 備抵呆帳：

備抵呆帳意指在應收帳款中**預估**收不回來的部份，歸類為應收帳款的減項，是實帳戶科目

### 呆帳的處理大致分成兩種模式：


- 直接沖銷法： 在應收帳款真正確定無法收回時才認列，其缺點為：應收帳款**高估**、缺乏評價之**完整忠實表達**性、可**操縱**損益


- 備抵法： 於**銷貨發生年度**估計其應收帳款可能發生呆帳，於是**先行予以入帳， 並減少當期應收帳款變現價值**

### 應收票據定義：

票據是指發票人或付款人指定於特定日無條件支付一定金額之書面承諾，若 **主要營業活動而產生之應收票據，若折現之影響非屬重大，則不需計算現值** 。

#### 入帳金額之時點：

和應收帳款相同，應收票據於銷貨交易完成或完成服務時入帳。

#### 入帳之金額：

非營業活動產生者： 貸款給員工，出售廠房設備等等，必須到 **完成交易後** 收到票據時才能入帳。

### 應收帳款融資定義：
是指企業將賒消而形成的應收帳款有條件地轉讓給專門的融資機構，使企業得到所需資金。應收帳款也可以成為金融性商品。
